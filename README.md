# 🏦 Système de Gestion Bancaire

![Java](https://img.shields.io/badge/Java-17-blue)
![POO](https://img.shields.io/badge/POO-Héritage-green)

## 📝 Description
Application Java démontrant les concepts de Programmation Orientée Objet :
- Héritage (CompteBancaire → CompteCourant/CompteEpargne)
- Polymorphisme
- Gestion d'exceptions
- Collections Java

## ✨ Fonctionnalités
- ✅ Création de comptes bancaires (courant, épargne)
- ✅ Dépôts et retraits avec validation
- ✅ Transferts entre comptes
- ✅ Calcul d'intérêts pour comptes épargne
- ✅ Gestion des découverts autorisés

## 🛠 Technologies
- Java 17
- Collections Framework
- Programmation Orientée Objet

## 🚀 Exécution
```bash
javac CompteBancaire.java
java CompteBancaire
